== Description ==

Contact Form 7 Customizer is a powerful admin panel with essential options to customize forms made with "Contact form 7 Plugin". 
 
Adjust the aesthetic of the forms to fix perfectly with your theme design. 

Give a professional look to your site with this essential tool designed to improve the features of one of the most fantastic plugins here, "Contact Form 7"


Need an extra option? Ask for in the support tab.

**List of available Options:**

* Main Font Size/color/style
* Fields Font Size/color/style
* Custom Fields width.
* Dropdown Menu Font Size/color/style
* Button Font Size/color/style
* Fields Background Color/padding/border radius
* Button Background Color/padding/border radius
* Dropdown Menu  Background Color/padding/border radius


== Installation ==

1. Upload the files to the `/wp-content/plugins/contact-form-7-customizer/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Look for the tab named "Contact Form 7 Customizer" at the left side of the admin screen.
